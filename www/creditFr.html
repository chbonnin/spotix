Crédits

Cette application intègre les éléments suivants

  • bibliothèque Javascript de cartographie Leaflet
  • base de données géographique et rendu graphique OpenStreetMap
  • cadre applicatif Cordova avec les extensions :
    • Géolocalisation
    • Presse-papier
    • Nommage avec caractères accentués
    • Utilisation du navigateur
    • Liste blanche d'adresses internet
  • quelques icônes Glyphish
  • extension Javascript de traduction
